    method = req.method      # 'GET'
    
    # Route to appropriate logic
    if path == '/courses' and method == 'GET':
        # Execute backend logic

        return jsonify({'data': courses})

## ðŸ“Š 4. Complete Backend Logic Flow

### Authentication Flow:
```
1. Flutter App â†’ POST /auth/register â†’ Cloud Function
2. Cloud Function â†’ Firebase Auth.create_user()
3. Cloud Function â†’ Firestore.collection('users').set()
4. Cloud Function â†’ Return success response
```

### Data Retrieval Flow:
```
1. Flutter App â†’ GET /courses â†’ Cloud Function
2. Cloud Function â†’ Verify Firebase ID Token
3. Cloud Function â†’ Query Firestore collections
4. Cloud Function â†’ Format and return JSON data
```

### Profile Management Flow:
```
1. Flutter App â†’ PUT /auth/profile â†’ Cloud Function
2. Cloud Function â†’ Verify Authorization header
3. Cloud Function â†’ Update Firestore user document
4. Cloud Function â†’ Return updated profile data
```
